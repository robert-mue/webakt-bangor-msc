What is AKT?

AKT stands for the Agroecological Knowledge Toolkit. It is a methodology for collecting knowledge from local farmers about the agricultural and ecological aspects of their farming practices, in the former of statements. These statements are represented in a formal notation, using the Prolog logic-programming language, and brought together in a knowledge base.

The methodology is supported by AKT5, a software package which has been developed to allow statements to be entered using the formal notation, checked for syntactic correcness, added to a knowledge base, searched, and analysed using software tools.

The AKT project was started in 1996, and based in Bangor University, North Wales. More information on the project itself, and the AKT5 software, can be found here. The AKT5 software is available for free download from the AKT website.

What is webAKT?

The primary aim of webAKT is to bring the AKT5 software package online, so that it can be accessed using any web browser. The existing software is now some 25 years old, maintenance is due to tail off shortly, and the software is written in a specialised language (LPA WinProlog), which few programmers know.

webAKT will be standards-based, using pervasive web languages (HTML, JavaScript and CSS), and possibly a standard Prolog, thus ensuring that there will always be a large pool of programmers - for example, in the IT departments of international agricultural research institutes - who can maintain and extend it.

A secondary aim, but one that could turn out to be equally important, is to open the door to developing web pages which give particular views of knowledge bases. So, for example, a report on a particular knowledge base from one location can include extracts and analyses of the knowledge base itself, seamlessly included in the web page. This should make the AKT knowledge bases accessible to a much wider audience.

Demos

  • AKT5 walk-through demo
    This is intended to give an idea of what the web-based version of AKT5 will be like. It includes two walk-throughs, highlighted in yellow menu items and buttons: loading a knowledge base and exploring its Topic Hierarchy; and accessing and running one Tool.
  • The AKT5 dialogue windows
    This shows all the AKT5 dialog windows, re-implemented in HTML. These were generated automatically from the AKT5 WinProlog codebase. These are all included in the AKT5 walk-through demo, but are hidden until they are actually called into play be a user action.
  • The Atwima knowledge base
    This page illustrates how webAKT Tools (separate modules that can undertake some specific analysis of a knowledge base) can be seamlessly integrated into a conventional web page, in the form e.g. of a table, so that a reader of the page need not even be aware how the table got there.
